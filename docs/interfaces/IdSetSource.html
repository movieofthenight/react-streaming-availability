country

country: string

2-letter country code to display the streaming availability information accordingly.

E.g. if you set it to "us", the shows will be displayed with their streaming availability information in the US.

See

https://docs.movieofthenight.com/guide/countries-and-services for the list of supported countries.

ids: string[]

The ids of the shows to fetch.
